# T0N1N0 — Electronic Press Kit

Static, dependency-free EPK for T0N1N0.

## Local preview

```bash
python3 -m http.server 4173
```

Open `http://localhost:4173`.

## Vercel

Import the GitHub repository and choose **Other** as the framework preset. No build command, install command, output directory, or environment variables are required.

## Content checks before public launch
